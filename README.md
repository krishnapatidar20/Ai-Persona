## Installation

Clone the repository

```bash
git clone https://github.com/yourusername/ai-persona.git
```

Go inside the project

```bash
cd ai-persona
```

Install dependencies

```bash
npm install
```

Create a `.env` file

```env
VITE_GEMINI_API_KEY=YOUR_GEMINI_API_KEY
```

Run the project

```bash
npm run dev
```
